turicreate.sound_classifier.SoundClassifier¶

class turicreate.sound_classifier.SoundClassifier(state)¶

A trained model that is ready to use for sound classification or to export to CoreML.

This model should not be constructed directly.

See also

create

Methods

SoundClassifier.classify(dataset[, verbose, ...]) Return the classification for each examples in the dataset.
SoundClassifier.evaluate(dataset[, metric, ...]) Evaluate the model by making predictions of target values and comparing these to actual values.

SoundClassifier.predict_topk(dataset[, ...]) Return top-k predictions for the dataset.
